Извлечение сценария из EXE файла

Допустим Вы уже создали на основе файла сценария приложение MyApp.exe. Но возникла необходимость изменить сценарий, а он уже удален. В этом случае извлеките код сценария из MyApp.exe запустив его с ключем /pullout.

MyApp.exe /pullout

В каталоге будет создан файл MyApp.os.
