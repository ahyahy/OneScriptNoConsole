Запуск сценария командой с параметрами

Для запуска из командной строки исполняемому файлу OneScriptNoConsole.exe нужно передать два аргумента. Это путь до файла oscript.exe и путь до файла стартового сценария "*.os".

"...\OneScriptNoConsole.exe" /os="C:\Program Files\OneScript\bin\oscript.exe" /script="C:\888\os\Приложение.os"

Порядок чередования ключей "/os=" и "/script=" при этом роли не играет. Программа проверит существование переданных в ключах файлов и запустит сценарий на исполнение. Пути содержащие пробелы следует заключить в кавычки.

При возникновении ошибок запуска в директории файла OneScriptNoConsole.exe будет создан файл error.log.
