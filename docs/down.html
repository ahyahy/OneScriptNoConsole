Первая версия дизайнера вышла 21 мая 2024 г. с номером 1.0.0.0

Версия 1.2.0.0 - 

Описание
- Исправлена ошибка подключения пакетов OneScript. В файл настройки создания exe файла (settings.cfg) добавлен ключ - startFrom

Версия 1.1.2.0 - 

Описание
- Добавлен вывод сообщений при выполнения сценария в файл error.log.
- Добавлена возможность использования библиотек односкрипта из каталога lib.
- Для успешного прохождения проверки антивирусом удалена интеграция библиотек односкрипта в EXE файл.

Версия 1.0.2.0 - 

Описание
- Для успешного прохождения проверки антивирусом изменен алгоритм извлечения данных из EXE файла.

Версия 1.0.1.0 - 

Описание
- Учтено различное обозначение переноса строки в разных операционных системах
- Добавлен вывод ошибок выполнения скрипта в файл error.log
- Изменен ключ извлечения стартового сценария с /extract на /pullout
- Исправлена ошибка обработки директивы #Использовать

Версия 1.0.0.0 - 

Описание
